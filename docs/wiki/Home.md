## About This Website

The RARS website is built using [Hugo](https://gohugo.io/), a static site generator. This means:

- The website is made up of simple text files (Markdown)
- Changes are tracked using Git/GitHub
- When changes are made, the site automatically rebuilds

**You don't need to know how to code** to make most changes to the website. This wiki will guide you through the process step by step.

## Need Help?
